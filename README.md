# link-in-bio

A stylish personal landing page with links to your portfolio and social profiles.

## Features

- **Animated Background**: Uses Vanta.js (animated birds effect) with Three.js
- **Responsive Design**: Mobile-friendly with smooth animations
- **Glassmorphism UI**: Modern frosted glass effect on buttons and header
- **Multiple Variants**: Choose between different background styles:
  - `index.html` - Vanta birds animation (default)
  - `index-image.html` - Static image background
  - `index-video.html` - Video background

## Quick Start

1. Clone the repository
2. Open `index.html` in your browser
3. Customize the links and profile information

## Customization

Edit `index.html` to update:
- Profile picture: Change `./images/bg-2.jpg`
- Username: Update the `<h1>` text
- Links: Modify the URLs in the `<ul>` list
- Icons: Replace SVG files in the `images/` folder

## Files

- `index.html` - Main landing page
- `index.css` - Styling (glassmorphism, animations, responsive layout)
- `images/` - Profile picture and social media icons
